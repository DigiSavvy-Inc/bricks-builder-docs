# Bricks Builder Developer Documentation

This directory contains documentation specifically for developers working with Bricks Builder. These resources guide you through extending, customizing, and integrating with Bricks Builder at a code level.

## Contents

- Creating custom elements
- Creating custom dynamic data tags
- Code signatures and other advanced developer topics
- Best practices for development with Bricks Builder

If you're developing extensions, themes, or plugins for Bricks Builder, these documents will help you follow the established patterns and best practices.
